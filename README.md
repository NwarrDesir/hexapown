
# Hexapawn

Ce petit projet est un jeu Hexapawn en Python, avec une interface graphique (Tkinter) et une "IA" maison qui apprend au fil des parties.

## C’est quoi Hexapawn ?
Hexapawn, c’est un mini-jeu de plateau à la sauce échecs, sur 3x3 cases. Le but : amener un de ses pions de l’autre côté ou capturer tous ceux de l’adversaire. Ici, vous jouez contre l’ordi.

## Comment ça marche ?
L’ordinateur (appelé Melvine_IA) commence sans rien savoir. À chaque partie, il retient les coups qui marchent et évite ceux qui le font perdre. Plus vous jouez, plus il devient coriace !

Tout se passe dans une fenêtre Tkinter, avec des boutons pour voir l’évolution de l’IA ou réinitialiser sa mémoire si elle devient trop forte.

## Pour lancer le jeu

```bash
python hexapawn.py
```
